# Budil Decision Log

重要な判断を「いつ / なぜ / 何を見て / 次にどうするか」まで残すためのログです。

## 記録ルール

- 重要な判断をしたら残す
- なぜその判断をしたのかを書く
- 結果が分かったら追記する
- 当たった判断・外れた判断を学習材料にする
