# Budil Playbooks

## 6月なのに予約が少ない時

トリガー：

- 7日以内の予約が0〜2件
- エアコン繁忙期なのに問い合わせが少ない
- 広告を出しているのに反応が弱い

優先行動：

1. Google広告の表示・クリック・CTA確認
2. LPのLINE/予約CTA確認
3. 既存客へのLINE追客
4. 法人営業リストを10件確認
5. 洗濯機クリーニング投稿
6. 完全分解LPの訴求確認

## 空き日が3日以上ある時

優先行動：

1. 今日営業すべき会社を確認
2. 興味あり / 見積り中 / 日程調整中へ再連絡
3. 既存顧客へ季節提案
4. Googleビジネスプロフィール投稿
5. SNS投稿
6. 法人向け営業文面を作成

## Google広告で表示はあるがクリックが少ない時

見る場所：

- キーワード
- 広告文
- 地域
- 検索語句
- LPとの一致

次の一手：

- 広告文を見直す
- 地域名を強める
- 料金訴求を入れる
- 完全分解と通常洗浄を分けて考える

## Google広告でクリックはあるがCTAが少ない時

見る場所：

- LPファーストビュー
- 料金到達率
- CTA位置
- LINEボタン
- 電話ボタン
- 不安解消要素

次の一手：

- CTAを上に追加
- 料金の見せ方を確認
- 写真・口コミ・実績を強化
- スマホ表示を確認

## CTAはあるが問い合わせが少ない時

見る場所：

- LINE導線
- 予約導線
- 問い合わせ文面
- 価格への不安
- 対応エリア
- 作業中は電話に出られない旨の説明

次の一手：

- LINE相談を強める
- 写真を送ると早いと説明
- 予約前の不安をFAQで消す
- 電話よりLINEが確実な導線にする

## 問い合わせはあるが成約しない時

見る場所：

- 価格説明
- 完全分解と通常の違い
- 日程調整
- 遠方料金
- 追加料金説明
- 返信速度

次の一手：

- 見積りテンプレ改善
- 料金表を分かりやすくする
- 施工写真を添付
- 日程候補を複数出す

## 雨・湿気・カビが強い時

優先サービス：

- 洗濯機クリーニング
- 浴室
- レンジフード
- エアコン内部カビ
- 完全分解

投稿テーマ：

- ニオイ
- カビ
- 湿気
- 子どもがいる家庭の安心
- 洗濯物の生乾き臭

## 台風前後

台風前：

- 無理な予約を入れない
- 家族優先
- 外作業・移動リスクを確認

台風後：

- エアコン不具合
- 水漏れ
- 室外機まわり
- 湿気・カビ訴求
- 予約再調整

## 予約が埋まっている時

優先行動：

- 高単価メニューを優先
- 完全分解を優先
- 遠方低単価は慎重に判断
- 既存客を大事にする
- 無理に広告費を増やさない

## 法人営業を強める時

対象：

- 不動産
- 管理会社
- 民泊
- 美容室
- 整骨院
- 飲食店
- 学童
- 小規模施設
- 事務所

次の一手：

- 営業プリセット選択
- 営業文面生成
- 初回連絡
- 次アクション日設定
- 興味ありなら再連絡
- 見積り中なら日程提案

## 現場後の追客

作業後にやること：

- お礼LINE
- 口コミ依頼
- 次回提案
- 洗濯機やレンジフード提案
- 法人なら定期清掃提案
- Budilに最終連絡日を残す

## アナリティクス番頭チェック（Cursor Browser Automation）

トリガー：

- 週1回のアナリティクス確認
- Organic Search の伸び・落ちを把握したい
- Search Console でクリック0・表示多のクエリを洗い出したい
- 外部チェックに GA4/SC の状況を残したい

役割分担：

| 役割 | ツール |
|------|--------|
| 外部サービスを見る目 | Browser番頭 |
| その場の読み取り・検査員 | Cursor Browser Automation |
| 記録・判断・行動候補化 | Budil |

手順（半自動・2026-06-25 検証済み）：

1. Cursor Browser Automation で GA4 を開く（ログイン済みタブ）
2. Search Console の検索パフォーマンスを開く
3. 画面上の数値のみ読み取る（推測しない）
4. `【Budil貼り付け用】` 形式に変換する
5. Budil 公開 URL（https://teruya1229.github.io/budil/）の外部チェック画面へ貼り付ける
6. 保存する
7. 最新レポート・履歴・ダッシュボードを確認する
8. 必要なら「行動候補に追加」または「今日やることに追加」

注意：

- 設定変更・保存・削除・送信はしない（読み取り専用）
- 外部チェックは売上確定ではない
- GA4 と Search Console の表示期間が一致しないことがある
- ログインは人間が先に行う

2. Browser番頭統合（専用 Chrome/CDP、`reports/*.md` 生成）
3. ローカル 1 コマンド化（PowerShell/bat）
4. Budil 本体ボタン化（Google API — 将来）
