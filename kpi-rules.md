# Budil KPI Rules

## 予約状況

暫定基準として記載してください（今後の実績に合わせて更新する）。

- 7日以内の予約 0〜1件：危険
- 7日以内の予約 2〜3件：注意
- 7日以内の予約 4件以上：通常
- 14日以内の予約 3件以下：営業強化
- 空き日が3日以上：営業・追客・投稿を優先

## 売上

- 今日の売上予定が0円：営業・追客優先
- 今週の売上見込みが低い：高単価メニュー提案
- 完全分解 / 複数台 / 法人案件は優先度高
- 遠方低単価案件は慎重に判断

## Google広告

- 表示あり / クリック0：広告文・キーワード・地域設定を確認
- クリックあり / CTA0：LP・CTA・ファーストビュー確認
- CTAあり / 問い合わせ0：LINE導線・問い合わせ導線確認
- 問い合わせあり / 成約0：価格説明・日程提案・返信文面確認
- 広告費だけ出て成約0：一時停止または改善判断

## LP

参考KPIとして以下を入れてください。

- 訪問 → 価格到達：80%目安
- 価格到達 → プレミアム/高単価説明到達：50%目安
- 訪問 → CTA：6%目安

判断：

- 価格到達が低い：ファーストビュー改善
- CTAが低い：CTA位置・料金・口コミ・写真を改善
- 完全分解LPでCTAが低い：不安解消と料金説明を改善

## 営業

v1.7の営業ルールと合わせてください。

- 次アクション日が今日以前：優先度高
- 興味あり：優先度高
- 見積り・提案中：優先度高
- 日程調整中：優先度高
- 初回連絡済み：優先度中
- 未営業：優先度中
- 成約：今日の営業から除外
- 見送り：今日の営業から除外
- 次アクション未設定：注意
- 最終連絡日が古い：再連絡候補

## サービス優先度

以下を整理してください。

### エアコンクリーニング

- 繁忙期の主力
- 予約が弱い場合は広告・LP・既存客追客を確認

### 完全分解

- 高単価
- 差別化しやすい
- 写真・実績・不安解消が重要

### 洗濯機クリーニング

- エアコン予約が弱い時の補助メニュー
- 湿気・ニオイ・カビ訴求と相性が良い
- SNS投稿向き

### ハウスクリーニング

- 競合価格が安い
- 無理に主力化しない
- 空き日対策として限定的に使う

### 法人営業

- 将来の安定化につながる
- 元請け化・定期化の可能性あり
- Budilで優先管理する価値が高い

## 危険サイン

- 予約が少ないのに営業先の次アクションが空
- 広告費が出ているのにCTAが無い
- 問い合わせがあるのに成約が無い
- 空き日が続くのに投稿・営業が止まっている
- LPや広告を変更したのに記録が残っていない
