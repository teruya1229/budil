# Budil デモ台本

## 3分デモ

1. **経営司令塔ホームを見る**
   - 今日の結論・最優先3つ・作業予定・受付・売上利益が一画面で分かることを説明
2. **今日やることを見る**
   - 優先タスクが自動で出ることを見せる
3. **受付から作業予定へ流す**
   - 受付・予約番頭で取り込み → 作業予定作成の流れを短く見せる
4. **売上・利益を見る**
   - 売上番頭と利益番頭で、数字の見え方を説明
5. **作業後フォローを見る**
   - お礼・口コミ依頼文の生成を見せる

**トーク例：**
> Budilは、現場業の受付・作業・売上・フォロー・利益・アクセス分析をまとめて見るための経営番頭です。会計ソフトではなく、毎朝5分で「今日は何を優先するか」を判断するためのツールです。

## 10分デモ

1. **デモデータ作成**（データ管理 → デモデータを作成）
2. **経営司令塔ホーム** — 毎朝の司令塔として全体像を説明
3. **受付・予約番頭** — AI番頭連携・受付取り込み
4. **予約・作業予定番頭** — 今日の作業・地図・カレンダー
5. **エリア番頭** — エリア別集計・遠方注意
6. **売上番頭** — 月間目標・達成率
7. **作業後フォロー番頭** — お礼・口コミ・リピート
8. **利益番頭** — 支出・粗利・改善ヒント
9. **アナリティクス番頭** — LP改善・広告判断
10. **経営レポート** — 週次・月次レポート出力
11. **データ診断** — 紐付け切れチェック・バックアップ
12. **1ヶ月無料体験案内** — sales.html または LINE CTA へ誘導

## 説明トーク例

```
Budilは、現場業の受付・作業・売上・フォロー・利益・アクセス分析をまとめて見るための経営番頭です。
会計ソフトではなく、毎朝5分で「今日は何を優先するか」を判断するためのツールです。

清掃・エアコン・便利屋など、現場で動くひとり社長の方が、
LINEや電話の受付、作業予定、売上登録、口コミ依頼、利益確認を
バラバラに管理しがちな課題を、一画面にまとめます。

まずはデモデータで全体の流れを見ていただき、
合いそうであれば1ヶ月無料体験から始められます。
```
